SuperCharge your site in seconds

== Description ==

SuperCharge is a platform that lets you install apps to your website instantly and enhance the user experience on your site. You can add Notification Bar, Lazy Loader, Page Progress, Google Analytics and many more... and all this, without writing a single line of code! No programming knowledge required.

To add any app to your website, at least minimal coding is required. However, SuperCharge is a platform that requires you to have zero coding skills. 
Customizing these apps is also easy. You simply need to select color choices, animation speed and styles available to allow easy branding. As a bonus, adding these apps will help increase website traffic.

== Installation ==

1. Upload `supercharge` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Your unique Site ID is generated automatically. You are ready to go!
